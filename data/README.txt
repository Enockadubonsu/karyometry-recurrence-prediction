# Data

The bladder karyometry dataset (`bladder.rds`) is not included
in this repository due to data use restrictions.

## Source

The data originate from:

> Bartels, P.H., et al. (1998). Karyometric analysis in the
> grading of bladder carcinoma. *Analytical and Quantitative
> Cytology and Histology*, 20(1), 35–43.

## Obtaining the data

The dataset is provided as course material for BIOS 648
(Statistical Learning) at the University of Arizona,
Department of Epidemiology and Biostatistics.

To reproduce the analysis, place `bladder.rds` in this
`data/` folder and update the file path in
`code/karyometry_analysis.R` line 18 accordingly.
